# dotfiles

Personal config files. Restore on any machine:

```bash
git clone https://github.com/majidraza1228/dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```

## What's included

| File | Destination | Purpose |
|------|-------------|---------|
| `.claude/CLAUDE.md` | `~/.claude/CLAUDE.md` | Global Claude Code instructions |
