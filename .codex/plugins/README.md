# Codex Plugins

Use this folder to document plugin standards for Codex-related tooling.

## Notes

- Codex plugins are packaged with a required manifest at `.codex-plugin/plugin.json`.
- Keep plugin-specific skills in a top-level `skills/` folder inside the plugin package.
- Keep MCP config in `.mcp.json` at the plugin root when needed.
- Use this repo as a source of plugin templates or packaging guidance, not as a cache of installed plugins.
