# Codex Skills

Store reusable Codex skills here.

## Standards

- Each skill lives in its own folder.
- Every skill must include `SKILL.md`.
- `SKILL.md` must define `name` and `description` in frontmatter.
- Use `references/` for supporting docs and `scripts/` for executable helpers.
- Keep skills narrow, reusable, and easy for Codex to trigger from normal task wording.

## Suggested core skills

- `fix-issue`
- `code-review`
- `implement-feature`
- `security-review`
